## 🚀 Quick Start

### 1. Install Node.js
Download from https://nodejs.org (LTS version recommended)

### 2. Install Tailwind CSS CLI

```bash
# Navigate into the project folder
cd luxe-fashion

# Install Tailwind CSS as a dev dependency
npm install

# OR install globally
npm install -g tailwindcss
```

### 3. Build the CSS

```bash
# One-time build
npm run build

# Watch mode (rebuilds automatically when you save files)
npm run watch
# OR
npm run dev
```

The compiled CSS will be output to `dist/output.css`.

### 4. Open in Browser

Open any HTML file directly in your browser:
```
src/pages/index.html
```

> **Tip:** Use VS Code with the **Live Server** extension for hot reload during development.

## ⚙️ How Tailwind CLI Works

```
src/input.css  →  [Tailwind CLI scans content paths]  →  dist/output.css
```

The CLI reads your `tailwind.config.js` to know:
1. **Which files to scan** (content paths) for Tailwind class names
2. **Custom theme** (colors, fonts, animations)
3. **Dark mode strategy** (`class` = toggle via `<html class="dark">`)

Only the classes you **actually use** end up in `output.css` — making it tiny in production.

## 🌙 Dark Mode

Dark mode is **class-based**. Toggle by adding/removing `dark` on `<html>`:

```js
document.documentElement.classList.toggle('dark');
```

The current theme is saved to `localStorage` and restored on page load via `main.js`.

In CSS, use `dark:` prefix:
```html
<div class="bg-white dark:bg-dark-800 text-stone-900 dark:text-stone-100">
```

## 🔍 SEO Features

Each page includes:
- `<title>` and `<meta name="description">`
- `<meta name="robots" content="index, follow">`
- `<link rel="canonical">`
- **Open Graph** tags (og:title, og:description, og:image)
- **Twitter Card** meta tags
- **Schema.org JSON-LD** structured data
  - `index.html` → `ClothingStore`
  - `blog.html` articles → `BlogPosting` + `author`
  - `about.html` team → `Person`
  - `product.html` → `Product` + `Offer`
- Semantic HTML: `<main>`, `<article>`, `<section>`, `<aside>`, `<nav>`, `<header>`, `<footer>`
- `aria-label`, `aria-current`, `alt` attributes throughout
- `itemscope` / `itemprop` microdata on key elements

## 🛠️ Development Tips

- **Add a new page:** Copy any existing page, update `<title>` and `aria-current` nav link
- **Add a new component:** Define it in `src/input.css` under `@layer components {}`
- **Add a new color:** Add it to `tailwind.config.js` → `theme.extend.colors`
- **Watch for changes:** Run `npm run watch` and edit any file — CSS rebuilds instantly
